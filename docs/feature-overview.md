# 機能説明（現行）

このドキュメントは、2026-04-25 時点でのアプリ機能を利用者向けに整理したものです。  
現行の主導線は `TODO管理` と `新版ジャーナリング（/v2/*）` です。

## 1. エントリー

- `/`
  - TODO管理とジャーナリングの入口
- ヘッダーの切替
  - `TODO管理` ボタン: `/todo`
  - `ジャーナリング` ボタン: `/v2/calendar`

## 2. TODO管理

対象画面:

- `/todo`

主な機能:

- タスク作成・編集・削除
- 今日 / 近日予定 / 完了 / 全タスク / ラベル / カレンダー表示
- ラベル作成・編集・削除
- タスク並び替え
- タイトル入力中の `@ラベル名` 補助

保存先:

- `localStorage` または backend API（設定で切替）

## 3. 新版ジャーナリング（思考振り返り PoC）

対象画面:

- `/v2/calendar`
- `/v2/day/:date`
- `/v2/day/:date/thinking`
- `/v2/week/:weekStart/thinking`

日次の流れ:

1. `/v2/day/:date` で「きっかけ + 自由記述」のメモカードを保存
2. ChatGPT 用プロンプトを出力
3. 返却 JSON を取り込み
4. `/v2/day/:date/thinking` で分類結果を確認
5. 問いに対する追記メモを保存（元メモとは別管理）

週次の流れ:

1. `/v2/week/:weekStart/thinking` で日次材料を集約
2. ChatGPT 用プロンプトを出力
3. 返却 JSON を取り込み
4. 週次自由記述メモを保存

## 4. 旧版ジャーナリングについて

旧版ジャーナリング（`/calendar`, `/day/:date`, `/week/:weekStart`, `/month/:yearMonth`, `/year/:year`）は削除済みです。  
現行のジャーナリング導線は新版（`/v2/*`）のみです。
